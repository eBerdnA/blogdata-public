 <p>Because I simply always keep forgetting how to un-/set proxy settings for Git this is my reminder for the future.</p>
<p><strong>Set proxy</strong></p>
<pre><code>git config --global http.proxy http://&lt;username&gt;:&lt;password&gt;@&lt;proxy-server-url&gt;:&lt;port&gt;
git config --global https.proxy http://&lt;username&gt;:&lt;password&gt;@&lt;proxy-server-url&gt;:&lt;port&gt;
</code></pre>
<p><strong>Unset proxy</strong></p>
<pre><code>git config --global --unset-all http.proxy # to remove it from the global config
git config --unset-all http.proxy  # to remove it from the local repo config file
</code></pre>
